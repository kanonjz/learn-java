用term可以进行精准匹配，如果要进行精准匹配在创建索引时就要设置成keyword。  
 the index types changed in elasticsearch 5.0. The string type has been replaced by the text (analyzed) and keyword (not analyzed) types. 
```
PUT /my_store 
{
    "mappings" : {
        "products" : {
            "properties" : {
                "productID" : {
                    "type" : "keyword"
                }
            }
        }
    }
}
```
