### 新建index
```
curl -X PUT 'localhost:9200/weather'
```

### 删除index
```
curl -X DELETE 'http://localhost:9200/weather'
```

### 查看所有索引
```
curl -X GET 'http://localhost:9200/_cat/indices?v'
```

## 新增记录
```
curl -X POST 'http://localhost:9200/students/class1' -d '
{
  "name":"小明",
  "sex":"男",
  "age":"15"
}'
```
返回结果：
```
